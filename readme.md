## 프로젝트 설명
> NFT 관련 마켓 서비스 제작하기 (제공되는 기능은 아래와 같다)
> 1. Kakao Klip 지갑 연동
> 2. 지갑 관련 정보 (보유 Klay, 보유 NFT) 가져오기
> 3. NFT 민팅하기
> 4. 보유한 NFT 마켓에 판매하기
> 5. NFT 마켓에서 판매하는 NFT를 구매하기
> 6. 모바일을 통한 접근 시 바로 Kakao Klip 연동됨
> 7. 웹을 통해 접근 시 QR CODE 출력을 통해 Kakao Klip 연동됨
